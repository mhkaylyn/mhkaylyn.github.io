TwoBits:
Mobile Application

ui/ux design and front-end development

TwoBits is a mobile application in the making that rounds up, to the nearest quarter or half-dollar, every credit and/or debit card transactions of the registered user and allows the user to donate the amounts to charities or political organizations of his/ her choosing. In this project, I created the wireframes and developed the high-fidelity UI prototype for the application. Along with another team mate, I also developed the front-end. Through this project, I was able to learn about version control, Agile project management with Scrum, web security, and Android application development using Android Studio.
